About TruthLens

TruthLens is an AI-Based Fake News Detection System developed to identify whether a news article is Fake or Real. The project combines machine learning with a user-friendly web interface so users can verify suspicious news content quickly and easily.

Model

Logistic Regression

Text Processing

TF-IDF Vectorizer

Backend

FastAPI + Python

Frontend

HTML, CSS

Project Overview

This system is designed to reduce the spread of misinformation by analyzing textual content from news articles. It studies word patterns, term importance, and trained model behavior to classify whether a news statement is trustworthy or misleading.

Key Features

  • Predicts whether a news article is Fake or Real.
  • Shows trending misinformation through a dedicated fake news page.
  • Includes article detail pages with category and result.
  • Supports search and category filter on the news page.
  • Provides a simple and user-friendly interface for quick checking.

Technologies Used

  • Python for machine learning and backend logic
  • FastAPI for routing and web application handling
  • HTML and CSS for frontend design
  • Scikit-learn for model training and prediction
  • TF-IDF for converting text into numerical features
  • JSON for storing fake news article data

How It Works

The user enters a news statement on the prediction page. The input text is first cleaned and transformed using TF-IDF vectorization. The trained Logistic Regression model then analyzes the processed text and predicts whether the article is Fake or Real.

Why This Project Matters

Fake news spreads quickly on social media and messaging platforms. This project aims to create awareness and provide a technical solution that helps users check suspicious claims before believing or sharing them.
